## English Version

MorseLink, developed by **BI4MOL**, is a professional-grade CW (Morse code) training and online communication platform. It is designed to reduce the entry barrier, improve training efficiency, and sustain long-term engagement through measurable progression.

The system integrates **structured stage-based training** with **real-time MQTT-powered online communication**, forming a unified loop of learning, feedback, and live simulation. A data-driven scoring model, rhythm analysis engine, and progressive unlock system enable continuous and quantifiable skill development.

### Core Value Proposition

* **Lower Entry Barrier**
  Structured onboarding designed for beginners to quickly build timing and symbol recognition.

* **Performance Feedback**
  Accurate accuracy metrics and rhythm evaluation for measurable improvement.

* **Real-time Online Interaction**
  MQTT-based communication architecture supporting live CW interaction.

* **Sustained Engagement Design**
  XP system, grading structure, and stage unlocking for long-term motivation.
