# MorseLink 小白快速入门（只看在线通联 + 训练）

> 适合第一次用 MorseLink 的用户  
> 目标：先能连上、先能发，再开始练

---

## 1. 开始前先做 3 件事（1 分钟）

1. 打开 `设置 -> 呼号和密码设置`，填好呼号和密码并保存。  
2. 打开 `设置 -> 服务器设置`，确认服务器地址和端口。  
3. 打开 `设置 -> 发射设置`，把 `WPM` 先调到你舒服的速度（建议新手 12~18）。  

> 【配图占位-快01】首次必做 3 项设置

---

## 2. 在线通联快速上手（2 分钟）

## 2.1 连接服务器

1. 点击菜单 `在线通联`。  
2. 在右侧点 `连接服务器`。  
3. 看到状态从“连接中”变成“已连接”即可。  

如果失败，先回到第 1 步检查呼号/密码和服务器设置。

> 【配图占位-快02】在线通联页 + 连接成功状态

## 2.2 调整频率

1. 在右侧 `频率控制` 区域拖动滑块。  
2. 先选一个你常用的频率（7000~7300 kHz）。  

## 2.3 开始发报

你有两种方式：

- 鼠标：按住 `点击发报` 按钮，松开结束。  
- 键盘：用“一般设置”里配置的发报键。  

发报时你会看到：

- 信号灯变化
- 可视化动画
- 发送电码和翻译内容更新

## 2.4 一键清屏

- 右侧点 `清屏`，可快速清掉当前收发显示。  

---

## 3. 训练快速上手（5 分钟）

## 3.1 先过新手教学

1. 点击菜单 `开始训练`。  
2. 第一次会进入新手教学页面。  
3. 建议点“完成教学并进入训练”；也可“本次跳过”。  

> 【配图占位-快03】新手教学页

## 3.2 进入 Unit 训练

1. 在课程地图里点 `开始训练`，或直接点可用的 Unit 卡片。  
2. 训练会自动在 Rx（听写）和 Tx（发报）步骤间切换。  

> 【配图占位-快04】课程地图 + Unit 卡片

## 3.3 Rx（听写）怎么做

1. 点“播放”听题。  
2. 在输入框写下你听到的内容。  
3. 点“提交”看反馈，再点“下一题”。  

## 3.4 Tx（发报）怎么做

1. 按住“按键发报”按钮（或用键盘）发出目标内容。  
2. 点“提交”看译码和评分。  
3. 点“下一题”。  

> 【配图占位-快05】Rx/Tx 训练进行页

## 3.5 看懂结算页

结算页重点看 3 个指标：

- 准确率
- 节奏评分
- 评级（A/B/C）

系统会提示你是“继续下一个 Unit”还是“建议再练一次”。

> 【配图占位-快06】结算页（通过示例）

---

## 4. 小白常见问题（只看最常用）

## 4.1 连不上服务器

- 没填呼号/密码  
- 服务器地址或端口错误  
- 网络不通  

## 4.2 按了发报没声音

- 检查系统声音输出设备  
- 检查 `设置 -> 一般设置` 里的蜂鸣开关  

## 4.3 训练总是过不了

- 先把 WPM 调低  
- 先保证准确率，再提速  
- 卡关就先“再练一次”当前 Unit  

---

## 5. 新手建议（最实用）

1. 每天练 10~20 分钟比一次练很久更有效。  
2. 先把“听清”和“发准”做到稳定，再追求速度。  
3. 在线通联和训练交替进行，进步最快。
